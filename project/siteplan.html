World of Warcraft Realms

SITE PURPOSE

The site provides a list of all of the Warcraft Realms, for Retail, Cataclysm Classic, and Classic Era realms, and includes filters to choose by realm, by region, and by localization, as well as some sorting features, such as sorting alphabetically and so on. The purpose is mainly an exercise in accessing a 3rd party API in demonstrating that I can build something with that API. Also, a way to extract the data.

Practical Usage #1:Useful site to extract all realm names for personal projects.

Practical Usage #2:Support site for generating auto-complete list for realm names for my GRM Program. Due to x-realms guilds, it is often challenging typing in complicated and lengthy realm names. Auto-complete is very helpful here, but first you need a table of realm names to extract from. Being able to query a current server list to provide to the GRM addon when updating would be immensely helpful.

Here's my GRM use-case! Would be nice to not have to manually maintain realm names.
